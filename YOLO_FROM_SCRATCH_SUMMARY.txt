3. Early Stopping
   ├─ Enabled with patience=50
   ├─ Stops if no validation improvement for 50 epochs
   ├─ Prevents overfitting
   └─ Saves time if converged early

4. Batch Size Tuning
   ├─ If OOM error: reduce to 8 or 4
   ├─ If underfitting: increase to 32
   ├─ Default 16: good balance
   └─ Larger = faster but less GPU efficient

5. Learning Rate
   ├─ Automatically handled by YOLO
   ├─ Uses cosine annealing schedule
   ├─ Starts high, decreases over epochs
   └─ No manual adjustment needed
