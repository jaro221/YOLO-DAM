WHAT WAS ADDED:
================================================================================

1. Hard Negative Mining Weight
   File: YOLO_DAM_loss_4tasks.py (lines 210-260)
   Function: hard_negative_mining_weight()

   What it does:
   - Identifies false positives (high prediction, low target)
   - Selects top 25% hardest negatives (hardest false positives)
   - Weights them 2x while others stay at 0.5x minimum

   Expected result: +3-4% F1 (better precision)

   Metric to watch: hard_neg_boost (should be 1.0-1.3)


2. Adaptive Focal Loss per Class
   File: YOLO_DAM_loss_4tasks.py (lines 263-308)
   Function: compute_adaptive_alpha()

   What it does:
   - Tracks precision per class (Crack, Foreign Particle, etc.)
   - Adjusts focal loss alpha based on class difficulty
   - Common classes get 0.25 alpha, rare classes get 0.50-0.75
   - Early epochs lighter, late epochs stronger focus

   Expected result: +2-3% F1 (better rare class detection)

   Metric to watch: adaptive_alpha (per-class values)


3. Curriculum Learning
   File: YOLO_DAM_loss_4tasks.py (lines 311-363)
   Function: curriculum_learning_weight()

   What it does:
   - Epoch 1: Focus on easy samples only (weight=1.0)
   - Epoch 300: Gradually include harder samples (weight 0.3-1.0)
   - Smooth progression based on prediction-target mismatch

   Expected result: +2-3% F1 (better convergence)

   Metric to watch: curr_weight_mean (should go 0.3 → 1.0)
